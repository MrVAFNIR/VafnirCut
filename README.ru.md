# ✂️ VafnirCut — Автоматическая нарезка тишины для Premiere Pro

**VafnirCut** — это бесплатный и "стильный" плагин для Adobe Premiere Pro, который автоматически вырезает тишину и вздохи из ваших видео и аудиодорожек с помощью нейросетей (на базе [auto-editor](https://github.com/WyattBlue/auto-editor)).

Вам больше не нужно тратить часы на черновой монтаж — плагин сделает это за вас в один клик!

## ✨ Главные фичи
- 🎙️ **Многоканальный анализ:** Выбирайте любые аудиодорожки (A1-A5) для поиска тишины.
- 🎛️ **Тонкая настройка:** Ползунки отступа (Margin) и порога шума (Threshold).
- 🎨 **Кастомный дизайн:** 6 крутых тем оформления (Forest, Ice, Lava, Clouds, Water, Earth) с интерактивными гейзерами и 1 строгая (Pro) для тех, кто любит минимализм.
- ⚡ **Полная автоматизация:** Установщик сам скачает нужную версию Python и настроит нейросеть.

## 📥 Как установить (Windows)

1. Перейдите в раздел **[Releases](../../releases/latest)** справа на этой странице и скачайте `VafnirCut_Installer.exe`.
2. Запустите файл **от имени Администратора**.
   > **⚠️ ВАЖНО:** Так как установщик автоматически настраивает системные папки Adobe и скрыто устанавливает Python, *Windows Defender или ваш антивирус могут выдать предупреждение*. Это нормально! Нажмите **«Подробнее» -> «Выполнить в любом случае»**. Исходный код открыт в этом репозитории
3. Дождитесь закрытия черного окна установки.
4. Перезапустите Premiere Pro.
5. Если Python был установлен в первый раз, то рекомендую перезагрузить ПК

## 🚀 Как пользоваться
1. Откройте Premiere Pro.
2. В верхнем меню выберите: `Окно (Window)` -> `Расширения (Extensions)` -> `VafnirCut`.
3. **Выделите клип** на таймлайне, который хотите нарезать.
4. Настройте порог тишины и выберите анализируемые дорожки в плагине (стандартные настройки уже готовы к работе).
5. Нажмите **«ОБРЕЗАТЬ ТАЙМЛАЙН»**.
6. Готово! Новая нарезанная секвенция появится в папке `CUT` в окне проекта.
